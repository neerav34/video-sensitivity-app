# Video Sensitivity Management App

Full-stack assignment implementation with:

- Backend: Node.js, Express, MongoDB, Socket.io, Multer, JWT
- Frontend: React + Vite

## Quick Start

### Backend

```bash
cd backend
npm install
cp .env.example .env
# edit .env with your values
npm run dev
```

### Frontend

```bash
cd frontend
npm install
cp .env.example .env
# edit VITE_API_URL to your backend URL (e.g. http://localhost:5000/api)
npm run dev
```

Then open the frontend dev URL in your browser (usually http://localhost:5173).
